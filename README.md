# Gerenciador de Tarefas - CRUD em C#

Projeto console desenvolvido em C# para gerenciamento de tarefas com operações CRUD e integração com SQL Server.

## Funcionalidades
- Adicionar tarefa
- Listar tarefas
- Editar tarefa
- Marcar tarefa como concluída
- Excluir tarefa

## Tecnologias
- C#
- .NET
- SQL Server
- Programação Orientada a Objetos

## Objetivo
Projeto criado para praticar conceitos de POO, CRUD, acesso a banco de dados e organização em camadas.
